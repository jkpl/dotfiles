# Files
;     open file
\e    open file in dir
\q    file browser
\/    search files
\x    close buffer
\b    go to buffer
gf    go to file under cursor

# Editing
C-/   comment line(s)
\c    copy to clipboard
\v    paste to clipboard
\rw   clear whitespace
cgn   replace searched

# Toggle \t
h     highlight        p     paste
n     line numbers     w     wrapping
c     cursorline       s     spelling

# Selection
+     expand
_     contract
gn    select searched

# Windows C-w
q     close window
w     next window
s     new horizontal
v     new vertical
o     close other windows
